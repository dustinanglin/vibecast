Support

dustin.anglin@gmail.com

Before you write

A few common questions:

  • A podcast won't import. Some shows are exclusive to specific apps and don't publish a public RSS feed. Vibecast can only subscribe to feeds that are publicly available. If you have the RSS URL directly, you can paste it into the search.
  • Audio stops in the background. iOS occasionally pauses audio when the system reclaims memory. If this happens repeatedly, please let me know which iPhone model and iOS version you're on.
  • Episodes aren't appearing after refresh. Pull down to manually refresh the subscriptions list. If a specific podcast isn't updating, opening its detail screen also triggers a fresh fetch.

What to include in a bug report

  • iPhone model and iOS version (Settings → General → About).
  • What you were doing when the bug happened.
  • What you expected versus what happened.
  • A screenshot, if relevant.

I read every email and try to respond within a few days.
